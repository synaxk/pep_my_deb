of various tool on debian 12.6 stable.
Usage:
	sudo ./pimp_my_debian.sh [options]

Options:
	-h, --help	Display this help page.
	-s, --standard	Install standard tools like Firefox, vim,..
	-i, --i3wm	Install the i3 window manager and other tools for customization.
	-p, --pentest	Install the tools for pentesting and forensic analysis.
